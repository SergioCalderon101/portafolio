Proyectos

A continuación te presento algunos de mis proyectos destacados:

Sistema de Gestión de Registros Académicos

Desarrollo de una plataforma que permite a los usuarios registrarse y acceder a un dashboard personalizado. Implementé el backend con Python (Flask) y MySQL, incorporando inicio de sesión seguro con bcrypt para el hash de contraseñas y autenticación de dos factores (2FA) con PyOTP. La interfaz fue construida con HTML, CSS y JavaScript. Además, configuré credenciales mediante variables de entorno (python-dotenv), e integré funcionalidades avanzadas como notificaciones en tiempo real y envío de correos electrónicos con Flask-Mail.

Synapse AI – Sistema de Chat Inteligente

Desarrollo de una plataforma web de chat inteligente que integra modelos de OpenAI (GPT-3.5, GPT-4, GPT-4o) para ofrecer respuestas contextuales y profesionales. Implementé el backend con Python (Flask) y un sistema robusto de logging con rotación de archivos para monitoreo continuo. La aplicación cuenta con historial persistente de conversaciones con metadatos JSON, selección dinámica de modelos de IA y un prompt de sistema altamente configurable para respuestas adaptativas. La interfaz fue diseñada con HTML, CSS y JavaScript, con un enfoque en un diseño moderno y responsivo. Además, integré manejo seguro de credenciales mediante python-dotenv, gestión de concurrencia con FileLock, soporte de CORS para integración con aplicaciones externas y generación automática de títulos de conversación.

Portafolio Web Profesional (SPA)

Portafolio profesional desarrollado como una Single Page Application (SPA) que integra HTML5 semántico, CSS3 responsivo y Bootstrap 4 para una presentación clara y adaptable. Utiliza JavaScript y librerías como jQuery, Owl Carousel, Animate.css, VenoBox y Typed.js para animaciones, efectos dinámicos y navegación fluida. El sistema incluye barra de navegación fija, scroll suave, galerías con lightbox, barras de progreso animadas y formulario de contacto validado con PHP. La estructura modular organiza eficientemente assets, librerías y plantillas, optimizando la carga con Font Awesome, Ionicons y un preloader. Este portafolio demuestra habilidades en desarrollo web moderno y diseño responsivo, sirviendo como plataforma profesional para mostrar perfil y proyectos en línea.
