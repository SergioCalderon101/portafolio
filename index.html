About Me

Hello, I'm Sergio Calderón, a Computer Systems Engineering student currently in my sixth semester with a GPA of 9.1.

I specialize in backend development and building scalable solutions, with practical experience in languages like Python and Java, as well as SQL databases. I also have knowledge in frontend (HTML, CSS, XML/QWeb) that complements my comprehensive approach to software development.

I worked as a IT Intern at a trading company, where I engineered internal workflows by customizing Odoo (XML views, Python, JavaScript, SQL) to streamline operations and improve data consistency. I built and launched a price-checker kiosk and a responsive product catalog with real-time inventory, accelerating lookups and improving both user and staff experience. Additionally, I sustained operations through technical maintenance and support (PCs, printers, cabling), produced Power BI and Excel reports for decision-making, and administered WordPress/cPanel to maintain a stable and scalable web platform.

My areas of interest are backend development, artificial intelligence, process automation, and technology ethics. I consider myself self-taught, disciplined, and results-oriented, always focused on learning and adding value to real projects.

Projects

Here are some of my highlighted projects:

Academic Records Management System

Development of a platform that allows users to register and access a personalized dashboard. I implemented the backend with Python (Flask) and MySQL, incorporating secure login with bcrypt for password hashing and two-factor authentication (2FA) with PyOTP. The interface was built with HTML, CSS, and JavaScript. Additionally, I configured credentials using environment variables (python-dotenv), and integrated advanced features such as real-time notifications and email sending with Flask-Mail.

Synapse AI – Intelligent Chat System

Development of an intelligent chat web platform that integrates OpenAI models (GPT-3.5, GPT-4, GPT-4o) to offer contextual and professional responses. I implemented the backend with Python (Flask) and a robust logging system with file rotation for continuous monitoring. The application features persistent conversation history with JSON metadata, dynamic AI model selection, and a highly configurable system prompt for adaptive responses. The interface was designed with HTML, CSS, and JavaScript, with a focus on modern and responsive design. Additionally, I integrated secure credential management using python-dotenv, concurrency handling with FileLock, CORS support for integration with external applications, and automatic conversation title generation.

Professional Web Portfolio (SPA)

Professional portfolio developed as a Single Page Application (SPA) that integrates semantic HTML5, responsive CSS3, and Bootstrap 4 for a clear and adaptable presentation. It uses JavaScript and libraries such as jQuery, Owl Carousel, Animate.css, VenoBox, and Typed.js for animations, dynamic effects, and smooth navigation. The system includes a fixed navigation bar, smooth scrolling, galleries with lightbox, animated progress bars, and a contact form validated with PHP. The modular structure efficiently organizes assets, libraries, and templates, optimizing loading with Font Awesome, Ionicons, and a preloader. This portfolio demonstrates skills in modern web development and responsive design, serving as a professional platform to showcase profile and projects online.
